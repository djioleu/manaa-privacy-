Politique de Confidentialité

MA NAA MEDIA  ·  Dernière mise à jour : 27 mai 2026

1. Identité du responsable

L'application MA NAA MEDIA est développée et exploitée par Bitsmakers.

Contact : brehmedjioleu@gmail.com

2. Données collectées

2.1 Données fournies par l'utilisateur

  • Adresse e-mail — lors de l'inscription ou via Google Sign-In
  • Nom d'affichage — saisi lors de la configuration du profil
  • Numéro de téléphone — optionnel, saisi dans les paramètres du profil
  • Photo de profil — optionnelle, choisie par l'utilisateur
  • Contenu publié — commentaires et messages dans le chat de l'Application

2.2 Données collectées automatiquement

  • Jeton de notification Firebase (FCM) — généré automatiquement pour l'envoi de notifications push
  • Données techniques anonymes — erreurs enregistrées à des fins de diagnostic

2.3 Données non collectées

Nous ne collectons pas : localisation GPS, historique de navigation, liste de contacts, identifiant publicitaire.

3. Finalité du traitement

DonnéeFinalité
E-mail, nomAuthentification et identification dans l'Application
TéléphoneContact optionnel (fonctionnalité future)
Photo de profilAffichage dans le profil et les commentaires
CommentairesAffichage public dans les sections interactives

4. Base légale du traitement

  • Exécution du contrat : traitement nécessaire pour fournir les fonctionnalités (radio, contenus, commentaires)
  • Consentement : pour les notifications push et les données optionnelles (téléphone, photo)
  • Intérêt légitime : sécurité, prévention des abus, amélioration du service

5. Partage des données avec des tiers

Nous ne vendons pas vos données. Certains prestataires techniques y accèdent dans le cadre strict de leur mission :

Supabase — Infrastructure backend

Base de données, authentification, stockage des fichiers médias.
Données transmises : profil utilisateur, commentaires, contenu uploadé.
supabase.com/privacy

Google Firebase — Notifications push

Envoi de notifications via Firebase Cloud Messaging (FCM).
Données transmises : jeton de notification (anonyme, sans données personnelles).
firebase.google.com/support/privacy

Google — Authentification

Connexion via Google Sign-In.
Données transmises : e-mail et nom fournis par Google lors de la connexion.
policies.google.com/privacy

Bunny CDN — Diffusion audio et vidéo

Hébergement et diffusion des fichiers audio (radio 24h/24) et vidéo.
Données transmises : aucune donnée personnelle — uniquement les fichiers médias.
bunny.net/privacy

YouTube — Diffusions en direct

Lecture des émissions TV et radio en direct via le lecteur YouTube intégré.
La politique de YouTube s'applique lors de l'affichage du lecteur.
policies.google.com/privacy

7. Conservation des données

  • Données de profil : conservées tant que le compte est actif. Supprimées sur demande dans un délai de 30 jours.
  • Commentaires : conservés dans la base de données. Supprimables par l'utilisateur ou un modérateur.
  • Jeton FCM : régénéré automatiquement par Firebase, non stocké de façon permanente par nos soins.

8. Droits des utilisateurs

Vous disposez des droits suivants :

  • Accès : obtenir une copie de vos données personnelles
  • Rectification : corriger des informations inexactes via les paramètres du profil
  • Suppression : demander la suppression de votre compte et de toutes vos données
  • Opposition : vous opposer aux notifications push depuis les paramètres de l'Application
Pour exercer vos droits, contactez-nous à brehmedjioleu@gmail.com. Nous répondons sous 30 jours.

9. Sécurité des données

  • Les communications sont chiffrées via HTTPS / TLS
  • Les mots de passe ne sont jamais stockés en clair (gestion Supabase Auth)
  • L'accès aux fonctionnalités d'administration est restreint par un système de rôles
  • Les fichiers médias sont servis via des URL sécurisées (Bunny CDN)

10. Enfants

L'Application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles les concernant. Si vous pensez qu'un enfant nous a fourni des données, contactez-nous à brehmedjioleu@gmail.com pour suppression immédiate.

11. Modifications de cette politique

Nous pouvons mettre à jour cette politique à tout moment. La date de « Dernière mise à jour » en haut de page sera modifiée en conséquence. Pour les changements significatifs, une notification sera envoyée dans l'Application.
